# HiveMind

万智牌轮抽工具

## 依赖

- Boost
- [Crow](https://github.com/ipkn/crow)
- [mstch](https://github.com/no1msd/mstch)

## 构建和运行

```
git clone https://github.com/mistivia/hive-mind.git
cd hive-mind
make && ./hivemind
```
